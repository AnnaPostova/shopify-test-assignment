# Featured Products Section

Цей репозиторій містить реалізацію кастомної секції **Featured Products** на базі стандартної теми **Shopify Dawn** у межах тестового завдання.

Секція показує рекомендовані товари з вибраної колекції та дозволяє додавати їх у кошик без перезавантаження сторінки.

---

## Функціональність

- Секція **Featured Products**
- Вибір колекції через Theme Customizer
- Товари, які вже є в кошику, не відображаються в секції
- Додавання товару в кошик через AJAX
- Відкриття та оновлення cart drawer теми Dawn
- Оновлення секції після додавання товару
- Адаптивна верстка (mobile-first)

---

## Виконані вимоги

### Основні
- ✅ Стандартна тема **Shopify Dawn**
- ✅ Адаптивна та кросбраузерна верстка
- ✅ Mobile-first верстка
- ✅ Додавання товару без перезавантаження сторінки
- ✅ Без jQuery
- ✅ Без CSS-фреймворків

### Додаткові
- ✅ BEM-неймінг для CSS
- ✅ SCSS
- ✅ Gulp для збірки стилів
- ✅ Компонентний підхід, як у Dawn
- ✅ Custom HTML Element (без Shadow DOM)
- ✅ Додавання товару в корзину оновлює та показує pop-up корзину теми Dawn.
- ✅ Додавання товару в корзину оновлює список товарів секції (оскільки доданий
продукт вже не показується в секції)ю
